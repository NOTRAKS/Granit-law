# Granite of Law

Профессиональные юридические услуги для бизнеса и частных лиц.

## Технологии

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS

## Разработка

```bash
pnpm install
pnpm dev
```

## Сборка

```bash
pnpm build
```

## Деплой

Проект автоматически деплоится на GitHub Pages через GitHub Actions при пуше в ветку `main`.
